With the helps of Hooks, Jenkins, Plugins & Github hook Trigger: once the Developer changed the code and commits:-
  • The Code got pushed to Github (with the help of Hooks)
  • Github alerts Jenkins about new code and ask to start the Job (using Webhook)
  • Jenkins goes to github and clones the new code to its workspace & deloys to the webserver. (with the help of Github Plugins)
  • Now the new Website is available to clients


All these above steps are automated and agile.
