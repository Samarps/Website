With the helps of Hooks, Jenkins, Plugins & Trigger: once the Developer changed the code and commits:-
  • The Code got pushed to Github (with the help of Hooks)
  • It is downloaded to the Jenkins workspace (when Jenkins finds a new version of code is available on github)
  • And then auto Deployment takes place (with the help of Jenkins)
  • Now the new Website is available to clients


All these above steps are automated and agile just by using a Jenkins Job.
